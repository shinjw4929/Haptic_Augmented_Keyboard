# 햅틱 증강 키보드를 통한 게이밍 경험 향상에 대한 연구  
### 본 연구는 PC환경에서 키보드와 마우스를 이용해 게임을 즐기는 게이머에게 햅틱 게이밍 경험을 제공할 수 있도록 한 연구이다.   
연구를 통해   
1. 햅틱 증강 키보드를 하드웨어적으로 구현하였다.
2. 게임 화면 분석을 통해 햅틱 피드백을 생성하는 소프트웨어적 방법론을 제시하였다.
